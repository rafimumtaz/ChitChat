# **App Name**: Chitter Chatter

## Core Features:

- Chatroom Creation: Allow users to create dynamic chatrooms with custom names and topics.
- Chatroom Joining: Enable users to join existing chatrooms via a search or invite.
- Friends List Management: Enable users to send, accept, and remove friend requests to manage their network.
- Real-Time Chat: Provide real-time messaging capabilities within chatrooms for seamless communication.
- UI Customization: Allow users to customize the UI with a color picker. The backend is providing an API using Python Flask/FAST API and the front end or the UI is a separate app that calls your Python backend

## Style Guidelines:

- Primary color: Soft blue (#A0C4FF) to create a calm and inviting atmosphere.
- Background color: Light gray (#F0F0F0) to provide a clean and modern feel.
- Accent color: Pale cyan (#BDB2FF) for interactive elements and highlights.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern, machined, objective, neutral look.
- Use simple, outlined icons to represent chatroom functions and user actions.
- Employ a clean, minimalist layout with clear separation of chatrooms, friends list, and message areas.
- Subtle transitions and animations to enhance the user experience when switching between chatrooms or updating the UI.
